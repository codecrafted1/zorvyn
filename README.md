FinBoard is a modern FinTech dashboard web application built using React.
It provides a clean and interactive interface to monitor financial data, track transactions, and visualize spending insights.

The project focuses on UI/UX design, interactive dashboards, and financial data visualization with a futuristic fintech-inspired theme.

Features
Dashboard
Summary cards displaying:
Total Balance
Income
Expenses
Time-based visualization (Balance Trend)
Categorical visualization (Spending Breakdown)
⚡ Smooth Navigation
Clicking the Dashboard card opens the full dashboard page.

Tech Stack
React.js
JavaScript (ES6)
CSS3
React Router
Recharts (for charts)
